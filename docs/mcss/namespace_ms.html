int Ms::transposeTpc(int tpc, Interval interval, bool useDoubleSharpsFlats)

Parameters
tpc The initial pitch spelling.
interval The interval to be transposed by.
useDoubleSharpsFlats Determines whether the output may include double sharps or flats (Abb) or should use an enharmonic pitch (Abb = G).
Returns The transposed pitch spelling (tpc).

Transposes a pitch spelling given an interval.

This function transposes a pitch spelling using first a diatonic transposition and then calculating any accidentals. This insures that the note is changed by the correct number of scale degrees unless it would require too many accidentals.
